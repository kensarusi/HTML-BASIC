Formularios

Los formularios permiten al usuario enviar información, como en un registro o un formulario de contacto.

Etiquetas del formulario

  • <form>: contenedor de todo el formulario.
  • <fieldset>: agrupa campos relacionados con un borde visual.
  • <legend>: título del fieldset.
  • <label>: texto que describe un campo. Debe ir siempre con su input.
  • <input>: campo de entrada. Cambia según el atributo type.
  • <textarea>: área de texto para mensajes largos.
  • <select> y <option>: lista desplegable de opciones.
  • <button>: botón de acción.
Buena práctica: Siempre enlaza <label> con su campo usando for e id con el mismo valor. Esto mejora la accesibilidad.

Tipos de <input>

  • text: texto libre.
  • email: valida que sea un correo con @.
  • password: oculta los caracteres.
  • number: solo números.
  • date: selector de fecha.
  • radio: elige una opción entre varias.
  • checkbox: elige una o varias opciones.
  • file: permite subir un archivo.
  • submit: botón para enviar el formulario.
