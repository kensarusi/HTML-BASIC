Aprenderás a mostrar datos organizados en filas y columnas usando tablas HTML.

Tablas

Las tablas sirven para mostrar datos organizados en filas y columnas, como una hoja de cálculo.

Etiquetas de tabla

  • <table>: crea la tabla.
  • <thead>: agrupa la fila de encabezados.
  • <tbody>: agrupa las filas de datos.
  • <tr>: una fila de la tabla (table row).
  • <th>: celda de encabezado. Sale en negrita y centrada.
  • <td>: celda con datos normales.
<table>
  <thead>
    <tr>
      <th>Etiqueta</th>
      <th>Categoría</th>
      <th>¿Tiene cierre?</th>
    </tr>
  </thead>
  <tbody>
    <tr>
      <td><p></td>
      <td>Texto</td>
      <td>Sí</td>
    </tr>
  </tbody>
</table>
Buena práctica: Las tablas son para datos, nunca para organizar el diseño de la página. Para eso usa CSS.

Ejemplo funcional

Etiqueta Categoría ¿Tiene cierre?
<h1> - <h6> Texto Sí
<p> Texto Sí
<img> Multimedia No
<ul> Listas Sí
<form> Formularios Sí
<table> Tablas Sí
<br> Texto No
