Listas

Las listas sirven para organizar información como una serie de elementos.

<ul> y <li> — Lista sin orden (viñetas)

¿Qué es? Lista con viñetas. Se usa cuando el orden no importa. Cada elemento va dentro de <li>.

<ul>
  <li>HTML</li>
  <li>CSS</li>
  <li>JavaScript</li>
</ul>
  • HTML
  • CSS
  • JavaScript
Buena práctica: Solo puede tener <li> como hijos directos. Úsala cuando el orden no importa.

<ol> y <li> — Lista ordenada (números)

¿Qué es? Lista numerada. Se usa cuando el orden sí importa, como pasos de un proceso.

<ol>
  <li>Crear el archivo .html</li>
  <li>Escribir el código</li>
  <li>Guardar el archivo</li>
  <li>Abrir en el navegador</li>
</ol>
  1. Crear el archivo .html
  2. Escribir el código
  3. Guardar el archivo
  4. Abrir en el navegador
Buena práctica: Úsala para instrucciones, recetas o pasos donde el orden es importante.

<dl>, <dt> y <dd> — Lista de definición

¿Qué es? Lista de términos con su descripción.

  • <dl>: contenedor de la lista de definición.
  • <dt>: el término (definition term).
  • <dd>: la descripción del término (definition description).
<dl>
  <dt>HTML</dt>
  <dd>Lenguaje para la creación de páginas web.</dd>

  <dt>CSS</dt>
  <dd>Lenguaje para dar estilos a páginas web.</dd>

  <dt>JavaScript</dt>
  <dd>Lenguaje de programación para páginas web interactivas.</dd>
</dl>
HTML
Lenguaje para la creación de páginas web.
CSS
Lenguaje para dar estilos a páginas web.
JavaScript
Lenguaje de programación para páginas web interactivas.
Buena práctica: Ideal para glosarios, diccionarios o pares de pregunta-respuesta.
