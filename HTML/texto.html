Etiquetas de Texto

Estas etiquetas dan formato y significado semántico al texto de la página.

<h1> hasta <h6> — Títulos

¿Qué es? Etiquetas de encabezado. <h1> es el más importante, <h6> el menos.

<h1>Título principal</h1>
<h2>Subtítulo</h2>
<h3>Sección</h3>
<h4>Subsección</h4>

Título h1

Título h2

Título h3

Título h4

Buena práctica: Solo un <h1> por página. No saltes niveles (de h1 a h3 sin pasar por h2).

<p> — Párrafo

¿Qué es? Define un bloque de texto como párrafo. El navegador agrega espacio arriba y abajo automáticamente.

<p>Este es un párrafo de ejemplo.</p>
<p>Este es otro párrafo separado.</p>

Este es un párrafo de ejemplo.

Este es otro párrafo separado.

Buena práctica: No uses <br><br> para separar bloques de texto. Para eso existe <p>.

<span> — Contenedor en línea

¿Qué es? Envuelve una parte del texto para aplicarle estilo sin romper el flujo del párrafo.

<p>El semáforo está en <span style="color:red">rojo</span>.</p>

El semáforo está en rojo.

Buena práctica: Úsalo solo cuando no haya una etiqueta más semántica disponible como <strong> o <em>.

<strong> — Texto importante

¿Qué es? Marca texto como muy importante. El navegador lo muestra en negrita.

<p>Es <strong>obligatorio</strong> guardar el archivo.</p>

Es obligatorio guardar el archivo.

Buena práctica: Prefiere <strong> sobre <b>. <strong> tiene significado semántico (importancia), <b> solo es visual.

<em> — Énfasis

¿Qué es? Marca texto con énfasis. El navegador lo muestra en cursiva.

<p>Debes guardar el archivo <em>antes</em> de cerrar.</p>

Debes guardar el archivo antes de cerrar.

Buena práctica: Prefiere <em> sobre <i>. <em> tiene significado semántico (énfasis), <i> solo es visual.

<br> — Salto de línea

¿Qué es? Inserta un salto de línea dentro de un párrafo. No tiene etiqueta de cierre.

<p>Dirección: Calle 10 # 5-20<br>Bogotá, Colombia</p>

Dirección: Calle 10 # 5-20
Bogotá, Colombia

Buena práctica: No uses <br> para crear espacio entre párrafos. Usa CSS (margin) para eso.

<hr> — Línea horizontal

¿Qué es? Dibuja una línea horizontal para separar secciones de contenido. No tiene etiqueta de cierre.

<p>Fin de la primera sección.</p>
<hr>
<p>Inicio de la segunda sección.</p>

Fin de la primera sección.

Inicio de la segunda sección.

Buena práctica: Úsalo para separar temas diferentes, no solo como decoración.
