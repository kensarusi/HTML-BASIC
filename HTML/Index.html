¿Cómo usar este blog?

Cada página explica una categoría de etiquetas HTML. Para cada etiqueta encontrarás su definición, sintaxis, un ejemplo visual y buenas prácticas.

Categorías del blog

1. Etiquetas Estructurales

Le dan estructura semántica a la página: <header>, <nav>, <main>, <section>, <article>, <aside>, <footer>.

2. Etiquetas de Texto

Dan formato y significado al texto: <h1>–<h6>, <p>, <span>, <strong>, <em>, <br>, <hr>.

3. Listas

Organizan información en elementos: <ul>, <ol>, <li>, <dl>, <dt>, <dd>.

4. Formularios

Capturan datos del usuario: <form>, <input>, <label>, <select>, <textarea>, <button>, <fieldset>, <legend>.

5. Tablas

Muestran datos en filas y columnas: <table>, <thead>, <tbody>, <tr>, <th>, <td>.

6. Multimedia

Agregan imágenes, audio, video y contenido externo: <img>, <audio>, <video>, <source>, <iframe>.
